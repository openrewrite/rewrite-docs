# 8.8.1 release (2023-10-26)

## New Recipes

* org.openrewrite.maven.cleanup.ExplicitPluginVersion: Add explicit plugin versions to POMs for reproducibility, as MNG-4173 removes automatic version resolution for POM plugins.
